PHONEBOOK
=========

To use InstantObjects' mapping and persistence:

1. remove the point in front of "$DEFINE UseInstantObjects" clause, inside the
   PhoneBook.inc file;
2. change unit Brokers.pas, removing or including InstantObjects brokers;
3. add IOModel.pas as an InstantObjects' model unit
   (View menu item | InstantObjects Model Explorer | first button);
4. save all project files (File menu item | Save All).
