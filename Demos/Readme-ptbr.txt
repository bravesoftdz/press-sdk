PHONEBOOK
=========

Para utilizar mapeamento e persistência do InstantObjects:

1. remova o ponto em frente a cláusula "$DEFINE UseInstantObjects",
   no arquivo PhoneBook.inc;
2. altere a unit Brokers.pas, removendo ou incluindo brokers do
   InstantObjects;
3. inclua a unit IOModel como uma unit de modelo do IO
   (View | InstantObjects Model Explorer | primeiro botão);
4. Salve todos os arquivos do projeto (File | Save All).
