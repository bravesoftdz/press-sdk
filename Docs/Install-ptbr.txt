Para instalar o PressObjects, basta incluir os diretórios
($Press)/Source/Core e ($Press)/Source/Brokers na lista de paths de
bibliotecas.

O PressObjects possui atualmente apenas packages de Runtime, pois
ainda não foi implementada nenhuma funcionalidade para a IDE.

A única dependência atual é um broker de persistência, portanto é
necessário instalá-lo para persistir objetos e ler objetos
persistidos. Vide Persistence-ptbr.txt.
