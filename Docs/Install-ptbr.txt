Para instalar o PressObjects, basta incluir os diretórios
($Press)/Source/Core e ($Press)/Source/Brokers na lista de paths de
bibliotecas.

O PressObjects não instala componentes e, atualmente, não possui
packages para instalação. Futuramente serão apenas instaladas algumas
opções no menu da IDE de seu compilador.

A única dependência é um broker de persistência, portanto é necessário
instalá-lo para persistir objetos e ler objetos persistidos.
